# v0.1 Success Flow Manual Smoke Test

This script validates the exact v0.1 golden path:

`Spotify URL -> exclusion filter -> download -> mutagen tag verify -> .m3u8 export -> batch appears in LibraryView`

## Prerequisites

1. `spotdl`, `ffmpeg`, and `mutagen` installed in the same environment as the app.
2. Spotify credentials configured for spotDL (`--client-id`, `--client-secret`, or `--user-auth`).
3. A known Spotify playlist URL with at least 2 tracks.
4. App database reset or isolated test profile.

## Steps

1. Open **New Batch Wizard**.
2. Paste Spotify URL and choose a test profile.
3. Add exclusion text entry: `Artist Remove - Song Remove`.
4. Start batch.
5. Confirm logs show:
   - `spotdl save` completed and `.spotdl` file created.
   - exclusion engine removed expected track(s) and wrote sanitized `.spotdl`.
   - `spotdl <sanitized.spotdl>` download completed.
6. Open batch details and run metadata verification.
7. Confirm each downloaded track shows `tags_verified=true` and `art_embedded=true`.
8. Export playlist to `.m3u8`.
9. Inspect exported file:
   - starts with `#EXTM3U`
   - has `#EXTINF:<duration>,<artist> - <title>` for each included track
   - file paths resolve to downloaded files.
10. Return to **LibraryView** and confirm batch row is visible with `COMPLETE` (or equivalent healthy terminal state).

## Expected Result

- Batch appears in LibraryView.
- Excluded tracks are absent from downloads and playlist.
- Downloaded tracks pass mutagen verification.
- `.m3u8` output is UTF-8 and correctly formatted.
